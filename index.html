CSS

1-¿que es css?

"Cascading Style Sheets" (Hojas de Estilo en Cascada), es un lenguaje de programación que se usa para definir el diseño visual de un documento HTML o XML, como una página web. Permite controlar elementos como el color, la tipografía, el tamaño, la posición y el layout de los elementos de la página.

2-¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS se puede usar para estilos de texto muy básicos como, por ejemplo, cambiar el color y el tamaño de los encabezados y los enlaces.
algunos programas son firefox,edge,chrome etc

3-¿A qué se llama regla de CSS?

El CSS es un lenguaje basado en reglas: cada usuario define las reglas que especifican los grupos de estilos que van a aplicarse a elementos particulares o grupos de elementos de la página web

4-¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

La regla está compuesta por los siguiente elementos: Selector: selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos

). Para dar estilo a un elemento diferente, solo cambia el selector.
Declaración: aplica las propiedades que querés dar al elemento seleccionado en el Selector. En este caso hay una sola regla, pero puede haber más
Propiedades: es lo que te permite dar un estilo a un elemento HTML. En el ejemplo la propiedad es color que le dará color a las letras de los párrafos. Siempre las propiedades se ubicarán a la izquierda.
Valor de la propiedad: las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor se ubicará a la derecha.

5-¿Qué es un selector en CSS?

Un seleccionador de CSS es la parte de una regla CSS que describe qué elementos en un documento la regla coincidirá. Los elementos a juego tendrán el estilo especificado de la regla aplicado a ellos.

6-Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

UNIVERSAL :Nos permite aplicar estilos a todos los elementos HTML de un documento y se indica mediante el *.
ETIQUETAS:Aplicamos propiedades CSS a una etiqueta. La misma se identifica sin utilizar <>.
DESCENDENTE :Con este selector seleccionamos un elemento que está dentro de otro del mismo tipo. Estos seectores siempre están formados por dos o más selectores separados entre si. El último selector será el elemento HTML al que se le aplicará las propiedades de CSS.
DE CLASES :Los selectores de clases comienzan siempre por un . (punto). Los clientes web (navegadores) identifican este punto y lo distinguen del resto de los selectores.
ID :Este tipo de selector es igual al de clase, aunque lo aplicaremos a un único elemento HTML de nuestra página. Esto se debe a que el atributo ID de HTML identifica a un elemento HTML por medio de un nombre.

PROPIEDADES DE CSS

PROPIEDAD DESCRIPCION VALORES POSIBLES
color define color de texto color:red; color: rgb(34, 12, 64, 0.6); color: hsl(30, 100%, 50%, 0.6);
background-color Define el color de fondo de un elemento background-color: teal;color: white; background-color: rgb(153, 102, 153); background-color: #777799;
font-size Define el tamaño de la fuente del texto. p { font-size: xx-large } h1 { font-size: 250% } span { font-size: 16px; }
font-weight Define el grosor de la fuente del texto font-weight: normal; font-weight: 100;
font-style Define el estilo de la fuente del texto (por ejemplo, si es cursiva o normal). font-style: normal; font-style: italic; font-style: oblique;
font-family Define el tipo de letra a utilizar para el texto. body { font-family: "Gill Sans Extrabold", Helvetica, sans-serif } receipt { font-family: Courier, "Lucida Console", monospace }
text-align Define la alineación horizontal del texto text-align: start; text-align: "." center; text-align: inherit;
text-decoration Define la decoración del texto text-decoration: none; /* Sin decoración */ text-decoration: underline red; /* Subrayado rojo */ text-decoration: underline wavy red; /* Subrayado rojo ondulado */
text-transform Define la transformación del texto text-transform: capitalize; text-transform: uppercase; text-transform: none;
letter-spacing Define el espacio entre letras. letter-spacing: normal; letter-spacing: inherit; letter-spacing: 3px;
line-height Define la altura de línea del texto. /* Unitless values: usa esta cifra multiplicada por el tamaño de fuente del elemento */line-height: 3.5; /* valores */line-height: 3em;
